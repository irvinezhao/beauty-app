# 美丽小站 - 美容预约管理平台

一个专为美容行业设计的预约管理小程序，支持用户预约、商家管理、会员系统等功能。

## 功能特性

### 用户端
- 📋 项目浏览 - 瀑布流展示美容服务项目
- 📅 在线预约 - 日历选择时间段
- 💬 社区互动 - 发布帖子、评论交流
- 📔 美容日记 - 预约记录时间轴
- 👤 个人中心 - 头像修改、资料管理、充值、优惠券

### 商家后台
- 👥 用户管理 - 查看用户、储值余额、累计消费
- 📝 项目管理 - 添加/删除美容项目
- 📆 预约管理 - 查看预约日历
- 💰 充值管理 - 充值方案设置
- 🎫 优惠券管理 - 发放优惠券
- 🏷️ 打折配置 - 折扣设置

## 技术栈

- 纯前端：HTML + CSS + JavaScript
- 数据存储：Firebase Realtime Database（云端同步）
- 部署：GitHub Pages
