# 美丽小站 - 美容预约管理平台

一个专为美容行业设计的预约管理小程序，支持用户预约、商家管理、会员系统等功能。

## 功能特性

### 用户端
- 📋 项目浏览 - 瀑布流展示美容服务项目
- 📅 在线预约 - 日历选择时间段
- 💬 社区互动 - 发布帖子、评论交流
- 👤 个人中心 - 资料管理、预约记录
- 🔐 手机号登录 - 验证码登录

### 商家后台
- 👥 用户管理 - 查看用户、标签、储值余额
- 📝 项目管理 - 添加/编辑/删除美容项目
- 📆 预约管理 - 日历视图、设置服务时间
- 💰 充值管理 - 充值方案设置
- 🎫 优惠券管理 - 发放优惠券

## 技术栈

- 纯前端：HTML + CSS + JavaScript
- 数据存储：LocalStorage
- 无需后端，部署即用

## 快速开始

1. 克隆项目：
```bash
git clone https://github.com/irvinezhao/beauty-app.git
```

2. 用浏览器打开 `index.html` 即可使用

## 默认账号

- 商家后台账号：`admin`
- 商家后台密码：`beauty123`

## 页面结构

```
├── 项目    - 瀑布流展示美容项目
├── 预约    - 在线预约服务
├── 社区    - 用户交流社区
├── 记录    - 美容日记时间轴
└── 我的    - 个人中心
```
